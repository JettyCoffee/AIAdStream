# AIAdStream 技术文档

这里是 AIAdStream 的技术方案文档，按模块拆分。每个文档围绕一个核心问题展开：遇到了什么、有哪些可选方案、为什么选了这条路、最终效果怎么样。

## 文档导览

| 文档 | 内容 |
|------|------|
| [架构总览](./architecture.md) | MVVM 分层设计、数据流向、模块依赖、Swift 6 并发模型 |
| [卡片动态化方案](./card-system.md) | 三种卡片样式的组件拆分策略、列表滚动性能优化、手势冲突处理 |
| [数据层设计](./data-layer.md) | SQLite 本地数据库 + 种子导入机制、分页策略、跨页面状态同步 |
| [AI 集成方案](./ai-integration.md) | DeepSeek Function Calling 循环、流式 SSE 解析、工具定义与输出约束 |
| [视频播放器池](./video-player.md) | AVPlayer 对象池设计、外流/内流播放模式切换、资源回收策略 |
| [埋点统计方案](./analytics.md) | 事件模型设计、曝光统计口径、KPI 聚合与可视化 |

## 阅读建议

架构总览 → 卡片系统 → 数据层 → AI 集成。前三个是 App 骨架，理解了它们再看视频播放和埋点统计会轻松很多。

如果你只关心某个具体问题——比如你们也在用 SwiftUI + Function Calling——直接跳到你需要的文档就好，每篇都是自包含的。
